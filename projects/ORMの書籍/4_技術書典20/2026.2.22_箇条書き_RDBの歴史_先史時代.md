# 人間によるまとめ
## 1. RDBの歴史
- ここからは永続化レイヤー抽象化とORMの歴史を追いかけていきます。その前提になるのが、抽象化される対象である永続化レイヤーそのものと、抽象化の動機になるロジック側です。それぞれRDBとOOPに至るまでの歴史を、先史時代から追いかけていきます。
- そのつぎはそれらが合わさるところであるアプリケーションの歴史です。ここはおもに1950年台からおいかけます
- んでやっとORMの歴史。ORMは特殊な土台の上に立っています
- DDDの歴史は現代
- ではまず、すべての根源となる永続化レイヤーの歴史から見ていきましょう。
### 1-1. 先史時代・古代の永続化
#### 1-1-1. プリミティブな保存
- この書籍はBookathonというイベントで書かれたものをもとに作られています。このイベントは三日間で書籍を書くという狂ったイベントで、４０人の参加者が１３チームに分かれて書籍を書きました。
- そこに参加した私はその後、その時の原稿を活用し、『ORMの歴史』というニッチなジャンルで技術書典に参加することにしました。この書籍はだいたい2万文字強。薄い書籍になっています。
- 著者はWebエンジニアですが、我々エンジニアにとって、考えたことファイルとして保存することは日常的に行なっている行為です。そればかりか、プログラミングコードを書いて保存したり、スクリーンショットを撮っては保存したり、日常はctr + sに溢れています。
- こういった、情報をなんらかの記録媒体に保存する行為を、難しい言葉で永続化と呼びます。テストの解答用紙に自分の名前を書くのも永続化ですし、画用紙にクレヨンで絵を描くのも永続化です。私は幼い頃、自分の手のひらいっぱいにインクを塗りたくって実家の壁に手形を残したことがあります。今でも実家に戻ればその手形を見ることができますし、祖母の悲鳴のような声も思い出せます。
- 壁に手形を残すという行為について言えば、2026年の1月にインドネシア・スラウェシ島にて紀元前6万7000年のものと考えられる世界最古の洞窟壁画が発見されています。こちら私よりも工夫しており、ステンシルーー壁に手をつき、上からインクを吹きかけることで型取りをしたーー画の壁画です。これは先史時代のものですが、もう少し時代が降れば他にも例はたくさんあります。例えばトルコ南東部、紀元前9000年ごろにつくられたギョベックリ・テペ遺跡。これまで最古の文明だと思われていたメソポタミア文明よりも6000年近く昔の祭祀場であり、かつ狩猟採集民によるものです。高さ5.5m、重さ15tにもおよぶ柱は人間を模しており、至る所に動物を模った彫刻が彫り込まれています。このように、情報を保存するという行為はプリミティブなモノです。
- もう少し保存する対象を具体的な情報に近づけてみましょう。例えば商取引の記録。こちらは紀元前3000年のメソポタミア文明で、粘土板に楔形の文字を刻むことで記録を残しています。のちのウル第三王朝と呼ばれる時代（紀元前2100年ごろ）になると、9万5千枚におよぶ粘土板が出土しており、そこには商取引や行政管理の情報が残されていたようです。

	- 粘土板
	- パピルス

- これらの記録媒体が、いつ、どんな用途で使われたのかについてまとめておきましょう

## コンポ4「記録の用途」補強ネタ

テーマ: 人間はあらゆるものを書き残す

### 呪い・呪術
- エア・ナーシルへの苦情の粘土板（紀元前1750年頃、ウル出土）。銅の品質が悪いと顧客ナンニが商人エア・ナーシルに送ったクレームレター。大英博物館所蔵。インターネットで「世界最古のカスタマークレーム」としてミーム化している
- ギリシア・ローマの呪いの鉛板（defixiones）。薄い鉛板に呪文を刻み、釘で打ち付けて井戸や墓に投じた。バース（英国）の温泉遺跡から130枚以上出土。盗まれた手袋を返さない犯人への呪いなど、日常的な内容

### 私的通信
- クラウディア・セヴェラの誕生日招待状（西暦100年頃、ヴィンドランダ要塞出土）。ローマ帝国北辺の軍事要塞から出土した木片の手紙。女性による最古のラテン語の手書き文書とされる。「私の誕生日パーティーに来てね」という内容
- ↑のエア・ナーシルの苦情も私的通信に分類できる

### 文芸
- エンヘドゥアンナ（紀元前2285-2250年頃）。アッカド帝国サルゴン王の娘で、ウルの月神ナンナの大神官。世界最古の既知の著者。讃歌や詩を粘土板に残した
- イェール大学所蔵の料理レシピ粘土板（紀元前1750年頃、古バビロニア）。約40種類のレシピ。世界最古のレシピ集。煮込み料理が多い
- ギルガメシュ叙事詩: 12枚の粘土板に刻まれた叙事詩。ニネヴェのアッシュルバニパル図書館から出土

### 法律
- ハンムラビ法典（紀元前1754年頃）。高さ2.25mの閃緑岩の石柱に刻まれた282条の法典。条文例: 第108条「女酒場主が穀物でなく銀で勘定をつけた場合、穀物の価値より高くしたときは、その女酒場主を水に投じる」。商取引の規制が含まれている

### 書記の練習
- エドゥバ（eduba、「タブレットの家」）と呼ばれる書記学校がシュメールに存在。生徒が練習で書いた粘土板が大量に出土。サティリカルな文章「書記学校の日々（Schooldays）」には、遅刻して先生に棒で叩かれる書記の少年の話が残っている

### 宗教
- エジプトの死者の書（紀元前1550年頃〜）。パピルスの巻物に書かれた、死後の世界への案内書。アニのパピルス（紀元前1250年頃）が最も有名で大英博物館所蔵

## コンポ5「商業活動とデータ管理」補強ネタ

テーマ: 行政・商業記録は古代から体系的なデータ管理が行われていた

出典: Robert K. Englund, "Equivalency Values and the Command Economy of the Ur III Period in Mesopotamia"（zettel/1_literatures/ に全文あり）
[[『EQUIVALENCY VALUES AND THE COMMAND ECONOMY OF THE UR III PERIOD IN MESOPOTAMIA（ウル第三期王朝の経済）』]]
[[『EQUIVALENCY VALUES AND THE COMMAND ECONOMY OF THE UR III PERIOD IN MESOPOTAMIA（ウル第三期王朝の経済）』_翻訳]]

### ウル第三期王朝の規模感
- 紀元前2100〜2000年頃の約50年間
- 約96,000枚の公開粘土板テキスト、同数の未公開テキストが存在する。この時代だけで
- これらは国家アーカイブではなく、地方の個別世帯（oikos）の管理記録

### 労働者への配給（賃金）システム
- 依存労働者（guruš=男性労働者、geme2=女性労働者）は、国家の被扶養者（corporate slaves）
- 月給: 女性は月30リットルの大麦、男性は月60リットルの大麦
- 非熟練日雇い労働者は約3倍（月200リットル相当）だが、衣服支給や病気時の保障はなし
- これらの配給記録が「数千枚の粘土板」に残っている

### 管理階層の給与表（HSS 4, 4という粘土板）
- ギルス（現テロ）出土の配給記録
- 最上位管理者（šabra）: 年200グル = 月約5,000リットルの大麦
- 牛管理者20名: 各年20グル
- 主任簿記係: 年100グル
- 主任測量技師: 年100グル
- 穀物倉庫管理者: 年100グル
- 耕牛記録係: 年40グル
- 労働部隊管理者: 年100グル
- 農夫98名: 各年15グル
- 合計2,630グル
- 最上位管理者は最下位労働者の約25倍の収入。現代で換算すると約37.5万ドル程度

### 等価値（equivalency values）システム
- 銀1シェケル（約8.33g）= 大麦300リットル = 魚油30リットル = 澄ましバター10リットル = 健康な羊1頭
- すべての商品を銀に換算できる仕組みが存在した
- 粘土板上の記法: 「n1 X, ku3-bi n2」（Xがn1個、その銀はn2）
- 等価値の用途4カテゴリ:
  1. 原材料と加工品の変換（大麦→パン、羊毛→衣類。加工に必要な原材料量を暗黙的に記録）
  2. 共通単位への変換（異なる穀物を大麦に、あらゆる商品を銀に換算）
  3. 管理上の調整（加工ロス、税、労働者の休日の計算。男性は1/10、女性は1/6の休日）
  4. 労働生産性の監視（実際の作業を確認せず、成果物を数えて労働規範テーブルから必要労働日数を逆算）

### 交易代理人（damgar）の帳簿（TCL 5, 6056という粘土板）
- ウンマの総督世帯の交易代理人シェシュカラの年次会計記録
- 前年からの赤字繰越（銀4.5シェケル12グレイン）から始まる
- 借方（liabilities）: 世帯から代理人に渡された物品（魚、魚油、羊毛、ナツメヤシ、革製品、穀物）をすべて銀換算で記録
- 貸方（credits）: 代理人が世帯に納めた物品（銀、銅、ラード、干しブドウ、木製容器）を銀換算で記録
- 借方 - 貸方 = 新しい赤字（4.5シェケル → 61 1/6シェケルに増加）
- この赤字は翌年の粘土板に繰り越される。年度をまたいで会計が連鎖している
- **これは複式簿記ではないが、借方・貸方の概念と赤字繰越による年次会計チェーンが紀元前2050年頃にすでに存在している**

### ネタとして使えるポイント
- 96,000枚の粘土板 = 50年間の簿記記録。この時代の「データベース」
- 等価値システムは、異なる商品を共通の尺度（銀）で比較可能にした。これはRDBでいう型変換やデータの正規化に近い発想
- 労働生産性の監視方法: 成果物を数えて労働規範テーブルから逆算する。暗黙のテーブル参照
- 交易代理人の帳簿は「個別記録→集計→翌年繰越」のワークフローそのもの

## コンポ6「複式簿記」補強ネタ

テーマ: データの正しさを記録の構造で保証する仕組み

### なぜ14世紀のイタリアで生まれたか
- 12〜14世紀、ヴェネツィア・フィレンツェ・ジェノヴァが地中海貿易を支配
- 「コメンダ」という共同出資契約（12世紀ジェノヴァ、最古の記録は1156年）。一航海ごとに出資者と航海商人が利益を按分する仕組み。航海が終わるたびに収支計算が必要だった
- 信用取引、為替手形、銀行融資、保険など金融取引が複雑化し、単式簿記では対応できなくなった

### 単式簿記の限界
- 現金の出入りしか記録しない
- 銀行から借りても「現金が増えた」としか見えず、負債の増加が見えない
- 利益と損失を計測できない
- 不正の検出ができない（チェック機構がない）

### 複式簿記の最古の記録
- アマティーノ・マヌッチの帳簿（1299-1300年）: フランスのニームに拠点を置くイタリア商人ジョヴァンニ・ファロルフィ商会の帳簿。現存する最古の複式簿記の実例。ただし不完全
- ジェノヴァ市のマッサーリ（財務管理官）の帳簿（1340年）: 現存する最古の「完全な」複式簿記システムの記録。蝋、胡椒、絹、塩、砂糖、穀物の商品勘定を含む
- 複式簿記は単一の都市から生まれたのではなく、ヴェネツィア・ジェノヴァ・フィレンツェで1250-1350年の間に徐々に発展した

### ルカ・パチョーリと「スムマ」（1494年）
- ルカ・パチョーリ（1445/46/47-1517年）。トスカーナ州サンセポルクロ生まれのフランシスコ会修道士にして数学者
- 少年時代に画家ピエロ・デラ・フランチェスカに数学の指導を受ける
- 1494年ヴェネツィアで『算術・幾何・比及び比例全書（Summa de arithmetica）』を出版。イタリア語で書かれた最初の代数学の印刷本
- この中の第9章第11節「計算と記録の詳細について」（27ページ）がヴェネツィア式簿記法を体系的に説明した部分
- パチョーリは発明者ではなく体系化した人物。本人もスムマの中でそう明言している。「会計の父」と呼ばれるのは、既存の実践知を整理して印刷で広めたから
- レオナルド・ダ・ヴィンチとの関係: 1496年ミラノ公の宮廷で出会い、パチョーリがレオナルドに幾何学を教えた。共著『神聖比例論（De Divina Proportione）』（1509年出版）ではレオナルドが多面体の挿絵を描いた

### 借方と貸方の何が画期的だったか
- すべての取引を二面的に記録する。銀行から借りたら「現金が増えた」と同時に「借入金（負債）が増えた」と記録される
- 借方合計と貸方合計は必ず一致する。一致しなければどこかに記帳ミスがある。構造によるエラー検出
- 一箇所だけ数字を変えると全体のバランスが崩れるため、改竄が困難
- 利益と資本の区別が可能になった（入ってきたお金が「儲け」なのか「借金」なのかがわかる）
- 注意: 試算表の均衡はすべてのエラーを検出するわけではない（借方と貸方を同額で間違えた場合や記帳漏れは検出不能）。partial check

### 複式簿記がもたらしたインパクト
- ゲーテ『ヴィルヘルム・マイスターの修業時代』第1巻第10章（1795-96年）: 「複式簿記は人間精神のもっとも美しい発明のひとつである」。ただしこれはゲーテ本人の発言ではなく、小説の登場人物ウェルナー（商人キャラ）のセリフ
- ヴェルナー・ゾンバルト『近代資本主義』（1902年）: 「資本主義と複式簿記は絶対に切り離せない。両者の関係は形式と内容のそれである」
- マックス・ウェーバー: 近代資本主義の特質は「合理的計算可能性」にあり、複式簿記はその技術的基盤
- ゾンバルト vs ヤメイ（B.S. Yamey）の論争: ヤメイは「複式簿記が資本主義の原因というのは過大評価」と批判。ブライヤーは「複式簿記は資本主義の原因ではなく結果」と主張
- この論争は「技術が社会を変えるのか、社会が技術を生むのか」という普遍的な問い。ORMの歴史でも同じ構造がある
