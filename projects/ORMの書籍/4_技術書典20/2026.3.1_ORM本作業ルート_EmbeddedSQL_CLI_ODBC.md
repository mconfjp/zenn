# 壁打ち
壁打ち済み → permanentノート化したもの:
- [[ODBCは接続を透過的にしており、どのデータベースでもから一本でいけた]]
- [[CLI（Call Level Interface）は1989年当初オブジェクト指向プログラミングに対応していなかった]]
- [[IBMは階層型DBで圧倒的シェアを誇っていたがRDBは群雄割拠で接続の標準化が必要になった]]
- [[TopLinkはSmalltalkコミュニティから生まれた]]
- [[IBMは1996年にOTIを買収してSmalltalk技術を内製化した]]
- [[IMS（階層型DB）を使わずファイルアクセスしていた時代があった]]
- [[MOC_Embedded SQLの仕組み]]
- [[Embedded SQLはプリコンパイルするが、課題としてクエリ内容や実行計画を動的にできないという問題があった]]
- [[MOC_Embedded SQL→CLI→ODBCの流れ]]
