- 歴史
	- ORMの概要
		- ORMとはOOPとRDBのマッピングをするもの
		- コード上で現れるオブジェクトと永続化レイヤーで保存されるリレーションは別の形式で保存されている。この変換をするのがORM
	- ORM登場までの
		- 1880s: 機械式計算機
			- ハーマン・ホレリスによる機械式計算機
				- オフィスの機械化
			- プログラミング可能な電子コンピュータが出てくるまでのオフィスの仕事ぶりがどうなっていたかかく
		- 1950s: ソフトウェアの登場
			- プログラミング可能な電子コンピュータ
			- アセンブラとパンドラの箱
			- SAGE防空システムとソフトウェア・コントラクター
		- 1960s: ソフトウェア危機の時代
			- コーディングバイブルの歴史
			- 同時に60年代はタイムシェアリングが始まっている
				- [[OSという概念はタイムシェアリングシステムによるマルチユーザー概念から生まれている]]
				- OSの概念が生まれる
		- 1970s: GUI, OOP, OS
			- [[UNIXは1969年に開発され、1973年にC言語で書き直され移植性が高くなったことで普及した]]
			- [[GUIでOOPが発展するのは必然。GUIの元になった構想はOOPという言葉を生み出したアラン・ケイのダイナブックに（も）ある]]
			- 1969年にXerox（複写機メーカー）が未来のオフィス用テクノロジーを開発する目的でパロアルトリサーチセンターを設立する
			- 1973年にAltoの開発が始まる
				- これは1981年にStarとして一般向けに発売されるが、商業的には失敗した（クソ高かった）
		- 番外編: ソフトウェア産業とRDB
			- 1959年のデータバンク
			- 1969年にコッド博士の論文
			- 1970年代を通してのIBMの研究開発
			- 1979年にOracleのOracle2が初めての商用向けRDBとして登場
			- 1983年にIBMのDB2が登場し、COBOLと一緒に使われる
				- データベース抽象化ではないもの
					- ORMと比較可能なデータベース関連製品
			- なんかあのミニコンピュータ向けアプリケーション市場でのRDBがあったはず
		- 1980s~1990s: GUIコンピュータの登場とOOPの商用利用
			- 1981年Xerox Star、1984 Mac
				- こいつらもSmalltalkのGUIの影響を受けている
				- 一旦Smalltalkから離れて出てきたのが面白い
			- 1990年代初頭のIBMのSmalltalk（VisualAge）移行PRJ
				- GUIの移行需要が高まっていたし、社内ではClassCという内部開発言語でOOPをやっていた
				- 単なる採用言語の変更ではなく、開発環境の移行だった
			- Win95
				- GUIが一般に広がる
			- 1995年にJavaも登場
		- ORMの登場
			- 1994年に初めてのORM、そして2002年に初めてのOSS製ORMが登場する
	- ORMの歴史（新領域でのORMの発展とOSS文化、Oracle）
		- 1994年にTopLink for Smalltalk
			- データマッパー型
		- 1995年にMySQL1.0がリリースされる、この時代はまだOODBMSへの期待も大きかった
		- 1996年にTopLink for Java
		- （いつ？）RedHatがIPO
		- 2002年にTopLinkがOracle社に買収される
		- 2002年に初のJava向けOSS ORMであるHibernateが登場
		- 2003年MySQL4.0リリース、InnoDBの採用
		- 2004年JBross inc法人化
		- 2005年にPHP向けのORMが登場
		- 2006年にRuby on RailsにActiveRecordが組み込まれる
		- 2007年にはWebフレームワークにはORMが標準装備されるように
		- 2010年にSunがOracleに買収
